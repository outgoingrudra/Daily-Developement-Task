Block and Inline Elements

Block elements takes 100% width -> div , address , p ,pre ,form

Inline elements takes upto the width it needs -> span ,i,b ,button

An inline element can't contain a block element

Semantic elements

  - elements with some meaning . e.g - img , video , table

Non - Semantic elements

  - elements with no meaning . e.g - div , p ,

Buttons - 3 types - normal button , submit , reset
